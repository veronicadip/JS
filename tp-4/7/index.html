Lista de Comandos

Bienvenido a tu agenda, si deseas ejecutar alguna acción, seguí las instrucciones. Cuando quieras dejar de usar funciones, apreta el botón cancelar.

  • Añadir contacto: ingresá el valor "añadir", luego el nombre del contacto, y finalmente su número.
  • Ver si un contacto ya existe: ingresá el valor "existe y luego el nombre del contacto."
  • Mostrar contactos: ingresá el valor "listar".
  • Buscar contacto por nombre y mostrar su número: ingresá el valor "buscar" y luego ingresá el nombre del contacto.
  • Eliminar contacto: ingresá el valor "eliminar" y luego el nombre del contacto.
  • Ver si la agenda está llena: ingresá el valor "llena".
  • Para cuantos espacios libres le quedan a la agenda: ingresá el valor "libre".
